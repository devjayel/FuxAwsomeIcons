FuxAwsomeIcons Documentation

Getting Started

To use FuxAwsomeIcons in your project, include the following CDN links:

CSS

<link rel="stylesheet" href="https://jayel223.github.io/FuxAwsomeIcons/path/to/yourfile.css">

JavaScript

<script src="https://jayel223.github.io/FuxAwsomeIcons/path/to/yourfile.js"></script>

Usage

Basic Icon

To display an icon, use the appropriate class in an <i> element. For example:

<i class="fa-solid fa-house"></i>

Animations

Enhance your icons with animations by adding the relevant classes:

  • Beat: Makes the icon pulse at a steady rate. 
    <i class="fa-solid fa-house fa-beat"></i>
  • Beat Fade: Similar to Beat, but the icon also fades in and out. 
    <i class="fa-solid fa-house fa-beat-fade"></i>
  • Bounce: Makes the icon bounce up and down. 
    <i class="fa-solid fa-house fa-bounce"></i>
  • Fade: Fades the icon in and out. 
    <i class="fa-solid fa-house fa-fade"></i>
  • Flip: Flips the icon horizontally. 
    <i class="fa-solid fa-house fa-flip"></i>
  • Shake: Makes the icon shake side to side. 
    <i class="fa-solid fa-house fa-shake"></i>
  • Spin: Makes the icon spin in a clockwise direction. 
    <i class="fa-solid fa-house fa-spin"></i>
  • Spin Reverse: Makes the icon spin in a counterclockwise direction. 
    <i class="fa-solid fa-house fa-spin-reverse"></i>
  • Spin Pulse: Makes the icon spin with a pulsing effect. 
    <i class="fa-solid fa-house fa-spin-pulse"></i>

Styles

FuxAwsomeIcons are available in several styles. To use a different style, replace fa-solid with the desired style class:

  • Solid: fa-solid
  • Regular: fa-regular
  • Light: fa-light
  • Thin: fa-thin
  • Duotone: fa-duotone

For example, to use a duotone icon:

<i class="fa-duotone fa-house"></i>

Examples

Here are some examples demonstrating the usage of different styles and animations:

  • Solid with Spin Animation: 
    <i class="fa-solid fa-house fa-spin"></i>
  • Regular with Bounce Animation: 
    <i class="fa-regular fa-house fa-bounce"></i>
  • Light with Beat Animation: 
    <i class="fa-light fa-house fa-beat"></i>
  • Thin with Flip Animation: 
    <i class="fa-thin fa-house fa-flip"></i>
  • Duotone with Shake Animation: 
    <i class="fa-duotone fa-house fa-shake"></i>
